# Chrome Web Store privacy practices

Use this file as the source text for the Chrome Web Store Developer Dashboard
Privacy practices tab. Keep it aligned with the generated Chrome manifest and
with the public privacy policy.

Reference docs:

- https://developer.chrome.com/docs/webstore/cws-dashboard-privacy
- https://developer.chrome.com/docs/webstore/program-policies/policies
- https://developer.chrome.com/docs/extensions/develop/concepts/declare-permissions

## Single purpose

Gemini Power Kit adds productivity tools to Gemini conversations on
`gemini.google.com`, including conversation navigation, prompt assistance,
theme controls, and optional response-complete notifications.

## Permission justifications

### `storage`

The `storage` permission is required to save user preferences, such as enabled
features, notification settings, audio preference, foreground notification
behavior, and theme options. These settings are stored by the browser so the
extension can keep the user's chosen experience across browsing sessions.

Gemini Power Kit does not use this permission to collect, sell, or share chat
content or personal information.

### Host access: `*://gemini.google.com/*`

Gemini Power Kit runs only on Gemini. Its content scripts match
`gemini.google.com` so the extension can add its user interface inside the
Gemini page. The content script reads and updates the Gemini page DOM to provide
user-facing features such as the chat outline, quick quote controls, prompt
tools, theme controls, and notification content extraction.

The Chrome manifest also declares `*://gemini.google.com/*` as optional host
access for response-complete notifications. This optional host access is
requested together with optional `webRequest` only after the user explicitly
enables response-complete notifications, because Chrome requires host access for
WebRequest observation.

For response-complete notifications, DOM access is used only after the browser
network completion signal has fired. The extension asks the Gemini content
script for a short notification title/message, foreground state, and, for image
responses, an optional local image preview. The extension does not run on
unrelated websites.

### Optional `notifications`

The `notifications` permission is optional on Chrome and is requested only after
the user explicitly enables response-complete notifications. It is used to show
a system notification when a Gemini standard response or Deep Research report
has finished, to show a user-triggered test notification, and to clear or focus
the originating Gemini tab when a notification is clicked.

The permission is not used for advertising, tracking, analytics, or background
collection unrelated to the notification feature.

### Optional `webRequest`

The `webRequest` permission is optional on Chrome and is requested together
with `notifications` and Gemini host access only after the user explicitly
enables response-complete notifications. It is used to observe successful Gemini
request completion for:

- `StreamGenerate` standard response completion.
- Deep Research `batchexecute` lifecycle requests needed to avoid early or
  duplicate notifications.

Gemini Power Kit does not inspect request bodies or response bodies for this
feature. It classifies only the request URL path and query parameters needed to
identify the Gemini completion lifecycle, and it listens only while the
notification setting is enabled and permissions are available.

### Optional `offscreen`

The `offscreen` permission is optional on Chrome and is requested only when the
user enables the notification sound option. It is used to create a Chrome
Offscreen Document for local audio playback after a system notification has
successfully been created.

The offscreen document plays the bundled `notification.mp3` asset. It does not
load remote code, fetch remote audio, collect user data, or change whether the
visual notification feature works.

## Remote code

Select: `No, I am not using remote code.`

Gemini Power Kit does not execute remotely hosted JavaScript. Extension scripts,
styles, images, and the notification audio file are bundled with the extension
package or generated by WXT at build time.

## Data usage disclosure

The extension handles Gemini page content locally to provide its user-facing
features, but it does not transmit Gemini chat content, notification content,
image previews, or user settings to the developer's servers.

### Data types handled locally

- Website content: Gemini conversation DOM content is read locally for features
  such as chat outline, quick quote, and notification title/message extraction.
- User activity: Gemini request completion events, page visibility/focus state,
  and notification click events are used locally to decide whether to show,
  suppress, clear, or focus a notification.
- User-provided content: Gemini prompts and responses may be present in the DOM
  that the extension reads locally for the user-facing Gemini tools.

### Data types not separately collected or shared

Sensitive information may appear inside the Gemini page if the user enters it
there. Gemini Power Kit treats that content as website content handled locally
for the requested Gemini tools. It does not separately collect, transmit, or
share:

- Personally identifiable information.
- Health information.
- Financial or payment information.
- Authentication information.
- Location.
- Web browsing history outside `gemini.google.com`.

Gemini Power Kit does not sell user data, use user data for personalized ads,
or transfer user data to data brokers or advertising platforms.

### Limited use certification

Certify that:

- Data is used only to provide or improve Gemini Power Kit's user-facing
  features.
- Data is not sold or transferred for advertising.
- Data is not used or transferred for credit-worthiness or lending purposes.
- Humans do not read user data unless required by explicit user consent,
  security, or legal compliance.

## Privacy policy notes

The public privacy policy should state that Gemini Power Kit:

- Uses permissions only to provide or improve the extension's single purpose.
- Handles Gemini page content and settings locally in the browser.
- Does not transmit Gemini chat content, notification text, generated image
  previews, or extension settings to developer-controlled servers.
- Does not sell user data or use it for advertising.
- Requests Chrome notification permissions only after the user enables the
  response-complete notification feature.
- Requests Chrome offscreen permission only after the user enables notification
  sound.
