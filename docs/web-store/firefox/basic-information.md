# Firefox Add-on Basic Information

## Categories

Select these categories:

1. Alerts & Updates
2. Appearance
3. Tabs

Rationale:

- **Alerts & Updates**: Gemini Power Kit can show reply-complete system notifications.
- **Appearance**: The extension includes Gemini theme, color, wallpaper, and readability controls.
- **Tabs**: Smart tab titles and Library open-in-new-tab behavior help users manage multiple Gemini sessions.

Do not select **Web Development** unless the listing is repositioned for developers. The extension is a Gemini productivity tool, not a web-development tool.
