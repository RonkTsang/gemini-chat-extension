Gemini Power Kit is a Firefox extension for Google Gemini that adds Chat Outline navigation, Quick Follow-up actions, Chain Prompt automation, visual themes, smarter tab titles, Library new-tab opening, and reply-complete notifications.

It is built for people who keep Gemini open for research, writing, coding, planning, and daily productivity. Gemini Power Kit removes repetitive steps inside Gemini so you can stay focused on the conversation instead of managing the interface around it.

## Key Features

### Quick Follow-up

Select text in a Gemini response and ask a follow-up without copy-paste.

- Ask about selected text directly from the page.
- Create reusable prompt shortcuts such as summarize, explain, translate, or rewrite.
- Run saved follow-up prompts in one click.

### Chat Outline

Navigate long Gemini conversations without endless scrolling.

- See a structured outline of prompts in the current chat.
- Jump directly to earlier parts of a conversation.
- Keep the outline pinned during long research or writing sessions.

### Chain Prompt

Turn repeated multi-step tasks into reusable prompt workflows.

- Build prompt chains for research, drafting, analysis, or review tasks.
- Use variables to reuse the same workflow with different inputs.
- Track execution progress step by step.

### Visual Themes

Customize Gemini for long working sessions.

- Choose appearance modes and color presets.
- Add wallpapers and readable glass-style surfaces.
- Keep the interface comfortable in both light and dark use.

### Reply-Complete Notifications

Step away while Gemini works in the background.

- Get a system notification when a Gemini reply is ready.
- Return to the originating conversation from the notification.
- Supports standard replies and completion paths such as generated media or Deep Research when Gemini exposes the final result state.

### Workflow Helpers

Gemini Power Kit also includes smaller quality-of-life improvements for daily use.

- Smart tab titles help distinguish multiple Gemini conversations.
- Library items can open in a new tab so you do not lose your scroll position.
- Settings are available directly inside the Gemini page.

## Privacy

Gemini Power Kit is designed for local use. Extension settings, prompt workflows, theme preferences, and UI state stay in your browser. The extension does not sell user data, use user data for ads, or transmit Gemini chat content to the developer or third parties.
