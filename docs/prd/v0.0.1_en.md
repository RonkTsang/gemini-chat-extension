**Title:** Chrome Extension for Gemini Chat History Index

**Background:**
When using gemini.google.com, Google's AI chat interface, it becomes difficult for users to locate specific past questions and answers in lengthy conversations. The goal is to create a browser extension that generates a quick index of the conversation history on the left side of the page.

**Task:**
Analyze the structure of the gemini.google.com webpage to extract the content of `user-query` elements. Use this content as titles for a quick index. Clicking on an index title should scroll the page to the corresponding DOM node.

**Example HTML Structure:**
```html
<infinite-scroller>
  <div>
    <user-query>
    </user-query>
    <model-response>
    </model-response>
  </div>
  ...
</infinite-scroller>
```
