## Chrome Web Store

You can install the extension directly from the Chrome Web Store:

[**Add to Chrome**](https://chromewebstore.google.com/detail/ihakfpnmefdkllhkecanagmienfnmojn)

## Install from Source

If you prefer to load the unpacked extension from source:

1. Clone or download our repository from GitHub.
2. Open Google Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode** using the toggle in the top-right corner.
4. Click the **Load unpacked** button.
5. Select the directory where you cloned the repository.

Once installed, the Gemini Power Kit icon will appear in the top-left corner of your Gemini chat window.
