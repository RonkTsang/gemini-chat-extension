## Choose by Task

Use this page to pick the feature that matches what you are trying to do. Each feature guide has a 30-second quick start.

## Feature List

| Feature | Best For | Where It Appears |
| --- | --- | --- |
| Chat Outline | Navigating long conversations | Sidebar on Gemini chat pages |
| Quick Follow-up | Running repeat prompts on selected text | Selection menu in Gemini chat |
| Chain Prompt | Multi-step workflows and automation | Sidebar in Gemini |
| Theme | Personalizing the Gemini interface | Extension settings panel |
| Open in New Tab | Opening items from "My stuff" without losing scroll | "My stuff" page |
| Tab Title Sync | Identifying tabs by chat title | Browser tab bar |

## New User Recommendations

1. Start with **Chat Outline** to avoid getting lost in long chats.
2. Add **Quick Follow-up** next to speed up repeat tasks.
3. Use **Chain Prompt** when you are ready to automate multi-step work.
