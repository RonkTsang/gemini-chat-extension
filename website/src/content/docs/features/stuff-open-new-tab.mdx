description: Smooth resource navigation in Gemini.

Open in New Tab solves the workflow interruption when browsing your historical resources in the "My stuff" page.

## Features

- **New Tab Entry**: Adds a dedicated button to every item in the "My stuff" page to open it in a new tab.
- **Scroll Preservation**: Keep your position in the stuff library while opening specific chat histories separately.
- **Seamless Integration**: Discreet buttons that appear on hover, maintaining Gemini's native aesthetic.
- **Workflow Continuity**: Switch between historical resources without losing current page state.
