## Overview

Open in New Tab adds a direct new-tab action to each item in **My stuff**.

## Why It Matters

On Gemini **My stuff**, opening a history item normally redirects the current page. This breaks browsing flow:

- You lose list context when leaving the page.
- Returning with Back often does not restore the previous scroll position.
- Native one-click new-tab entry is missing on item cards.

This feature solves those gaps by adding a dedicated **Open in new tab** action where users actually browse history.

## Key Capabilities

- Open a history item without leaving the current list page.
- Keep your current scroll position while exploring multiple items.
- Compare several chats/files across tabs in parallel.

## Quick Workflow

1. Go to **My stuff** in Gemini.
2. Hover an item card.
3. Click **Open in new tab**.
4. Continue browsing and opening more items.
