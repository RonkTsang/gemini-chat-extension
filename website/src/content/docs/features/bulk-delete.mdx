## Clean Up Gemini Chats Without Repeating the Same Delete Flow

**Bulk Delete** adds multi-select controls to the Google Gemini chat sidebar, so you can remove several unwanted conversations in one deliberate pass instead of opening each chat menu separately. It is designed for cleaning up drafts, one-off research, test prompts, or a long list of finished conversations while keeping the chats you still need easy to find.

The feature works on `gemini.google.com` in the **Recents** section. It does not remove anything until you select chats, press **Delete**, and confirm Gemini Power Kit's warning.

## Where to Find Bulk Delete

In the Gemini sidebar, find **Recents** and select the small trash icon beside its heading. Gemini Power Kit opens Bulk Delete mode directly in the sidebar and adds a checkbox to each available chat.

## How to Delete Multiple Gemini Chats

1. Open [gemini.google.com](https://gemini.google.com).
2. In the Gemini sidebar, expand **Recents** if needed and select the **Bulk Delete** trash icon.
3. Select chats individually, or choose **Select recent**.
4. For a recent batch, select 10, 20, 30, or 50 under **Change**. Turn on **Exclude pinned** first when you want to preserve pinned chats.
5. Check the number on **Delete**. Use **Deselect pinned** or **Deselect all** if you need to adjust the batch.
6. Select **Delete** and confirm the irreversible action.
7. Wait for the progress indicator to finish. The tool returns to Bulk Delete mode with no chats selected, ready for another review.

## When Bulk Delete Is Useful

- **After prompt experiments:** remove throwaway conversations created while comparing prompts, models, or outputs.
- **After research sprints:** clear finished searches and short investigations without disturbing your pinned reference chats.
- **Before sharing your screen:** reduce sidebar clutter and remove private, irrelevant, or duplicate chat titles from view.
- **For regular history maintenance:** select the newest 10–50 chats, exclude pinned work, and keep Recents focused on active conversations.

## Turn Bulk Delete On or Off

Bulk Delete is enabled by default. To change that setting:

1. Open [gemini.google.com](https://gemini.google.com).
2. In Gemini's left navigation, select **Gemini Power Kit** above **Settings & help** to open the extension settings panel.
3. Under **Tools**, open **Enhancements**.
4. Turn **Bulk Delete** on to show the Recents trash-icon entry, or off to remove the entry and its sidebar controls.

Turning the setting off does not delete Gemini chats. It only disables Gemini Power Kit's Bulk Delete interface. Turn it back on in the same place whenever you need it again.

## Important Notes

- Bulk Delete only acts on chats you explicitly select in the current session.
- A confirmation is required before deletion starts.
- Use **Exclude pinned** before batch selection when you want to protect pinned conversations automatically.
- If Gemini cannot load all requested recent chats, select again or reload the Gemini page before deleting.
- Gemini's chat interface can change over time; keep Gemini Power Kit updated for the latest compatibility fixes.
