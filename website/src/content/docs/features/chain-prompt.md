Chain Prompt elevates Gemini usage from simple chats to true automation by linking multiple prompts together.

## Features

- **Multi-Step Automation**: Run conversation workflows sequentially without manual intervention.
- **Variable Support**: Define dynamic parameters and variables for your workflows.
- **Visual Editor**: Build and manage your complex chain prompts through an intuitive graphical interface right inside Gemini.
- **Real-time Monitoring**: Track execution progress with live status updates in the sidebar.
