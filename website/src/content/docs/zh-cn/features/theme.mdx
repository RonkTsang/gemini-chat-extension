“主题个性化”允许您通过一系列精选的颜色预设和背景选项来打造个性化的 Gemini 界面。

## 功能特性

- **颜色预设**：从多套内置颜色主题中选择，匹配您的个人风格。
- **实时预览**：在设置面板中即时查看更改效果，满意后再全局应用。
- **平滑切换**：优雅的 UI 过渡效果，让主题切换更加自然。
- **自动持久化**：您的主题选择会自动保存，并可在不同设备间同步。
