## Chrome 应用商店

您可以直接从 Chrome 应用商店安装扩展：

[**添加到 Chrome**](https://chromewebstore.google.com/detail/ihakfpnmefdkllhkecanagmienfnmojn)

## 从源码安装

如果您希望从源码加载未打包的扩展：

1. 从 GitHub 克隆或下载我们的仓库。
2. 打开 Google Chrome 并导航到 `chrome://extensions`。
3. 使用右上角的开关启用**开发者模式**。
4. 点击**加载已解压的扩展程序**按钮。
5. 选择您克隆仓库的目录。

安装完成后，Gemini Power Kit 图标将出现在 Gemini 聊天窗口的左上角。
